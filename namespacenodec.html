◆ type_id()

template<typename Type >
std::enable_if_t< std::is_same_v< Type, std::remove_cv_t< std::remove_reference_t< Type > > >, const type_info & > nodec::type_id ( )
noexcept

Returns the type info object associated to a given type.

If type is a reference type, the result refers to a type_info object representing the referenced type (that is, type_id(T&) == type_id(T)). In all cases, top-level cv-qualifiers are ignored by type_id (that is, type_id(const T) == type_id(T)). These behaviors are same as native typeid operator.
