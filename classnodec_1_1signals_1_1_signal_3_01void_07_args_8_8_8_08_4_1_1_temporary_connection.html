Detailed Description

template<typename... Args>
class nodec::signals::Signal< void(Args...)>::TemporaryConnection

The temporary connection object. Operations on connections are possible only after they are assigned.

Warning
Holding the temporary object for a long time mat destroy the connection it holds. If you want to use a connection, you should assign it to the connection object as soon as possible.
